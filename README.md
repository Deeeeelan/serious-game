# serious-game

### aka. Cognitive Stress

A simple tower defense game with gears! Gather resources by throwing your tools, craft them into a network of cogs, and use them to power your towers to defend your core to survive the week.

Submission to The Very Serious Juniper Dev Game Jam!<br>
